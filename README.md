# Alfalah Rice Traders - Landing Page

A professional, responsive single-page website for **Alfalah Rice Traders SMC Private Limited**, a premium rice supplier and exporter.

## 🌾 Project Overview

Modern landing page showcasing premium rice varieties, quality standards, and providing an integrated contact form for quote requests. Built with clean HTML5, CSS3, and vanilla JavaScript - no frameworks, fully responsive.

## ✨ Features

- **Fully Responsive** - Optimized for mobile, tablet, and desktop
- **9 Key Sections** - Hero, Stats, About, Products, Quality, Testimonials, Contact
- **6 Rice Varieties** - Basmati, Jasmine, Brown, Steamed, Premium White, Super Kernel
- **SEO Optimized** - Meta tags, Open Graph, semantic HTML
- **Accessible** - WCAG AA compliant with keyboard navigation and ARIA labels
- **Performance Focused** - Lazy loading, minimal JavaScript, optimized assets

## 🚀 Quick Start

1. **View the landing page:**
   - Open `doc/index.html` in any modern browser
   - Or use a local server: `python -m http.server 8000`

2. **Customize:**
   - Edit content in `doc/index.html`
   - Change colors in `doc/styles.css` (CSS custom properties)
   - See full customization guide in `doc/README.md`

## 🎨 Design System

- **Colors:** Green (#2d5016) and Orange (#f4a259) with neutral grays
- **Typography:** Inter (body) + Playfair Display (headings)
- **Spacing:** 8px base unit system
- **Responsive:** Mobile-first with breakpoints at 640px, 768px

## 🛠️ Technologies

- HTML5 (semantic markup)
- CSS3 (custom properties, Grid, Flexbox)
- Vanilla JavaScript (no dependencies)
- Google Fonts
- Responsive images
